Instructions

Text File

The text file must have the ".txt" extension. Within the text file, use the following symbols to indicate the starting configuration:

"R" - starting position of the robot
"S" - location of the storage units
"O" - locations of a single obstacle (wall)
"." - indicates that a robot and storage unit are in the same location
" " - empty space where robot could possibly move (not a storage or an obstacle and does not contain currently contain a robot or a box)
"\n" - line break, place at the end of each line
